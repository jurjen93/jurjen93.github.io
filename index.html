Merging galaxy clusters (A399-401)

I lead a study about the merging clusters A399 and A401. We created the deepest low-frequency radio map and used this to study the re-acceleration of the cosmic rays in the radio bridge between the galaxy clusters. In our paper we argue that second-order Fermi re-acceleration is currently the most favored process to explain the radio bridge. Image credits: F. Sweijen.

Software

Open source

When I write code, I like to publish open source versions of my code, such that others can use it, contribute, or play around with it.

Benford's law

I wrote my first Python package based on code from my master thesis project. With this package you can determine how 'Benford' your data is, which means how well your data follows Benford's law. Check it out here or install it from the command line with pip install benfordslaw-analysis.

LOFAR helpers

While working with the Low Frequency Array (LOFAR), I wrote code to improve the data reduction and calibration. These turned into lofar_helpers. The script that has been used most from this library is h5_merger.py. This code has been implemented in the LOFAR facet-calibration pipeline and VLBI pipeline. With this script you can merge calibration solution files in H5parm format in Python or from the command line.

Walk-through astronomy videos

For a press release I wrote some code to make videos of our produced radio maps by inserting an image. This code will walk through your image automatically, or the user can specify a specific path. You can find the code here and a video that I created with it here.

Redshifting

For one of my projects I wrote code to simulate how a radio galaxy looks like if you move it to a higher redshift (which means moving to a farther distance). This helps to study the cosmic evolution of radio sources. You can install the code from the command line with pip install redshifting, or find it here.
